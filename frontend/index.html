Running Notes

Running Notes is an inbox for thoughts you capture on the go. Record a voice note in your browser and it appears as an audio attachment in a dedicated mailbox, ready to review here or in an IMAP client such as Outlook, Apple Mail, or Thunderbird.

  1. Sign in, or try the public guest account.
  2. Record a voice note.
  3. Process it using familiar inbox tools.

The current version stores recordings as they are; it does not transcribe or summarize them.

Guest privacy: The guest mailbox and its recordings are public. Do not use it for private or sensitive information. Guest recordings expire automatically after 24 hours.

Manual setup

  1. Add an IMAP account and use the displayed email address as the username.
  2. Incoming mail uses port 993 with SSL/TLS.
  3. Outgoing mail uses port 587 with STARTTLS.
  4. Generate a new app password here if you do not have one.
Outlook for Android troubleshooting Advanced

Outlook may retain an incorrect automatically detected server even when the settings above are correct. If it selects the wrong server, clear Outlook's application data and configure the account again using the displayed settings.

  1. Close Outlook.
  2. Open Android Settings → Apps → Outlook → Storage & cache.
  3. Choose Clear storage or Clear data, then reopen Outlook and add the account again.

Warning: This removes every Outlook account and all locally stored Outlook settings from the device. Mail stored on the servers is not deleted.
